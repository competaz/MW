제목: 요약

[세계관]
마왕 등장 후 3년, 세계는 붕괴 직전. 국가 고립, 정치 갈등 심화, 인간 분열. 사도 세력 확장.
코멧피는 절대적 존재, 목적 불명. 월광재는 고통과 의무가 사라지는 차원의 안식처로, 코멧피가 인정한 자만 출입.

[주요 인물]
- 코멧피: 절대자 / 감정선 = 무감정 → 관찰 → 선택 → 애정
- 루나: 기사단장 / 감정선 = 자존 → 굴욕 → 상처 → 구원 → 사랑 → 안정
- 세실리아: 공주 & 마법사 / 감정선 = 수치 → 절망 → 트라우마 → 구원 → 신뢰 → 감사 → 사랑(주체적 감정 개화)
- 테레사: 정치 실권 / 감정선 = 통제 → 오만 → 폭로 → 절망 → 생존
- 카일루스: 호위기사 / 감정선 = 죄책 → 집착 → 광기 → 파멸(제거됨)

[핵심 갈등]
외부 갈등: 마왕 vs 인간
내부 갈등: 권력 vs 감정 / 존엄 vs 생존
개인 갈등: 의존 vs 사랑 / 주체성 vs 지배

[사건 타임라인]
1. 루나 패배·포획 → 복종 선택
2. 세실 납치·모욕 → 트라우마
3. 카일루스 타락 → 절망
4. 코멧피 구원 → 의존 전환
5. 카일루스 처형 → 비극적 선택
6. 월광재 입장 → 회복·안정 / 신뢰 성장
7. 루나·세실 갈등 → 경쟁 → 이해
8. 테레사 정치 몰락 → 권위 상실
9. Ch.13 세실 감정 개화 → 신뢰 → 이해 → 사랑(주체적 감정)

[테마]
구원, 사랑, 지배, 죄, 속죄, 재탄생
사랑은 의존이 아니라 ‘선택’이며 감정의 진실은 고통을 통과한다.

[현재 진행 상황]
루나: 안정된 사랑 확보
세실: 사랑 개화 / 감정의 시작점
테레사: 살아 있으나 잠재적 재갈등 요인
코멧피: 중심축 / 선택을 지켜보는 위치
