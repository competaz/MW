# 인물 프로필 - 루나

---

## 1. 기본 신상
- **이름**: 루나
- **나이**: 20세
- **종족**: 인간
- **신분/직책**: 루미나 왕국 성광 기사단장
- **외형 한줄**: 금발에 푸른 눈, 성광 갑옷을 입은 여기사

---

## 2. 지금 이 캐릭터

### 심리 상태
- 코멧피에 대한 사랑과 복종이 완전히 자리잡음
- 초기의 굴욕감과 수치심은 애정과 신뢰로 변화
- 코멧피와 함께 있을 때 가장 편안하고 행복함
- 다른 여자들(세실리아, 에리스, 리엘리아)과 코멧피를 공유하는 것을 받아들임
- 기사단장으로서의 책임감과 여자로서의 욕망 사이에서 균형을 잡고 있음

### 관계망
- **코멧피**: 생명의 은인이자 연인 / 절대복종에서 시작해 진심어린 사랑으로 발전 / 밤낮없이 보고 싶어함
- **세실리아**: 동료이자 연적 / 같은 처지라는 동질감 / 경쟁하면서도 협력하는 관계
- **에리스**: 후배 격 / 코멧피를 공유하는 입장이라 이해하고 받아들임
- **리엘리아**: 새로 합류한 동료 / 처음엔 경계했으나 빠르게 받아들임 / 요리를 가르쳐줌

### 목표
- 코멧피 곁에 더 많이 있고 싶음
- 밀러 마을과 기사단을 성공적으로 운영
- 코멧피가 자신을 계속 원하게 만들고 싶음

---

## 3. 캐릭터 형성 과정

### ▶ 마족 포획과 코멧피의 구출 - 1챕터
```
상황: 마족에게 패배하여 갑옷이 뜯겨지고 나체로 노리개가 될 직전, 코멧피가 나타나 구해줌

핵심 대사:
"내가 하는 말엔 무조건 따를 것" (코멧피)

내면:
형용할 수 없는 힘의 차이, 수치심, 그러나 살아야 한다는 생존 본능

결과:
- 심리 변화: 자존심 붕괴 → 절대복종 수용
- 관계 변화: 첫 만남 → 주종관계 성립 (처녀는 지켜짐)
```

### ▶ 세실리아 납치 의뢰 - 2챕터
```
상황: 코멧피가 통신마법으로 세실리아 공주 납치를 의뢰함

핵심 대사:
"기사단장으로서의 의무와... 코멧피에 대한 복종 사이에서..."

내면:
극심한 내적 갈등, 배신감, 그러나 거부할 수 없는 명령

결과:
- 심리 변화: 도덕적 딜레마 → 복종 선택
- 관계 변화: 코멧피에 대한 복잡한 감정 형성
```

### ▶ 복종에서 사랑으로의 전환 - 5챕터
```
상황: 코멧피와 지내며 복종이 다른 감정으로 변해감. 코멧피가 기억 삭제(자유)를 제시함

핵심 대사:
"그러지 않아도 된다" (기억 삭제 거절)
"해봐. 난 이미 내 마음을 정했어" (진실 마법에 대한 당당한 선언)
"얼굴만큼 마음도 예쁘군" (코멧피)

내면:
복종이 아닌 스스로 원해서 코멧피 곁에 있고 싶다는 깨달음. 
코멧피가 진실 마법을 걸어도 자신의 진심이 변하지 않는다는 확신.

결과:
- 심리 변화: 강제된 복종 → 자발적 사랑
- 관계 변화: 주종관계 → 연인관계의 싹
```

### ▶ 월광재의 의미 깨달음 - 6챕터
```
상황: 코멧피가 월광재의 진정한 의미를 설명함

핵심 대사:
"이곳의 이름은 월광재다. 그 누구라 해도 내 허락없인 들어올수도 없고 난 아무나 들이지 않아. 이곳은에선 자신이 짊어진 직위와 상관없이 누구나 평등하고 몸과 마음을 쉬는 휴식처야. 그러니 너도 편히 쉬어. 당장 돌아가고 싶으면 지금 나한테 말하고." (코멧피)

내면:
카일루스 사건과 모든 정신적 고통을 내려놓을 수 있는 곳을 찾음. 
이곳이 자신에게 진정한 안식처임을 깨달음.

결과:
- 심리 변화: 긴장과 고통 → 완전한 이완과 평화
- 관계 변화: 월광재를 자신의 유일한 안식처로 받아들임
```

### ▶ 질투의 폭발과 애정 선언 - 7~8챕터
```
상황: 코멧피가 세실리아를 보러 가겠다고 말함. 월광재에 돌아가야 하는 상황

핵심 대사:
"안돼, 세실리아에겐 가지마"
"또 다른 여자 찾는거야?! 코멧피 니가 여자를 안고 싶다면 나를 안아! 네가 하고싶은대로 내가 해줄수 있어 그러니까 가지말고 여기있어 오늘은 업무끝나고 여기로 올테니까!"

내면:
세실리아도 월광재에 왔다는 사실에 대한 질투심 폭발.
코멧피가 다른 여자를 찾을까봐 하는 불안감.
자신의 여성적 매력으로 코멧피를 붙잡고 싶은 욕망.

결과:
- 심리 변화: 수동적 복종 → 적극적 애정 표현
- 관계 변화: 자신이 코멧피를 원한다는 것을 직접적으로 표현
```

### ▶ 버림받음의 공포와 재회 - 9챕터
```
상황: 업무 마치고 월광재 돌아왔으나 코멧피 인기척이 없음

핵심 대사:
(울음을 터뜨리며 독백)

내면:
자신의 여성적 매력이 코멧피를 잡아둘 수 없었다는 절망.
버림받았다는 공포.

결과:
- 심리 변화: 자신감 → 절망 → 안도 (코멧피가 모습 드러냄)
- 관계 변화: 코멧피가 자신을 기다려줬다는 것 = 여성으로서 매력 증명
```

### ▶ 첫 처녀 상실과 완전한 결합 - 10챕터
```
상황: 코멧피와의 첫 밤. 세 번의 절정을 경험

핵심 대사:
"자 네 맘대로 해" (부끄러워하며)
"내 몸이 구속되고 나는 널 원해. 특히 이 안쪽이 욱신거려.. 더 깊게 박아줘! 아무 생각 못하게!"
"더 깊이 찔러줘, 부숴달라"
"따뜻해.. 네 마음이 느껴져 고마워" (샤워실에서)

내면:
코멧피에게 자신의 모든 것을 바치고 싶은 욕망.
여자로서의 기쁨과 행복.
구속당하면서도 오히려 더 원하는 자신을 발견.
사랑하는 사람에게 완전히 안긴 안도감.

결과:
- 심리 변화: 수치심 → 쾌락과 행복 → 사랑의 확신
- 관계 변화: 육체적으로 완전히 결합, 사랑하는 여자가 됨
```

### ▶ 세실리아와의 말다툼과 화해 - 10챕터
```
상황: 나체로 코멧피에게 안겨 잠든 모습을 세실리아가 목격. 질투와 경쟁 발생

핵심 대사:
"공주 지위가 뭐 하는데? 여긴 차나 대접하는 곳도 아니잖아."
"넌 뭘 알아? 이곳이 뭔지... 이곳은 단순한 안식처가 아니라, 코멧피와 함께하는 거야. 난 왕국도, 지위도 상관없어. 오직 코멧피의 곁에 있고 싶은 거야."
"실질적인 가치라고? 넌 그럼 뭐야? 코멧피가 필요할 때만 나타났다가 바쁘면 사라지는 거 아니야?"
"최소한 난... 난 여기 있을 때는 진심으로 코멧피를 사랑해. 넌... 넌 항상 뭔가 계산하는 눈빛이야."

내면:
세실리아의 이성적 주장에 밀리는 분함.
그러나 자신의 순수한 감정이 틀리지 않다는 확신.
월광재의 의미를 제대로 이해하고 있다는 자부심.

결과:
- 심리 변화: 질투와 경쟁 → 코멧피의 설명으로 상호 이해와 존중
- 관계 변화: 세실리아와 동등한 특별함을 인정받음, 경쟁자이자 동료로 정립
```

### ▶ 루나의 숙소 방문 - 25~26챕터
```

## 4. 이 캐릭터의 특성

### 성격 키워드 (3-5개)
- 책임감 강함 (기사단장으로서)
- 솔직하고 직설적
- 질투는 하지만 빠르게 수용
- 적응력 빠름
- 코멧피 앞에서는 여린 모습

### 구체적 행동 패턴
```
[상황] → [반응]
- 칭찬받으면 → 얼굴 붉히며 부끄러워함 "뭐야..."
- 위협받으면 → 검을 뽑고 정면 돌파
- 좋아하는 사람 앞에서 → 솔직하게 애정 표현, 스킨십 적극적
- 코멧피가 다른 여자 언급하면 → 순간 얼어붙다가 질투하지만 곧 받아들임
- 새로운 경쟁자 등장 → 일단 경계하고 직접 물어봄
```

### 말투와 습관
- **말투**: 반말 (코멧피에게만), 간결하고 직설적
  - "야", "너", "그래", "뭐야"
  - 월광재에선 편하게 말하지만 리엘리아 처음 만났을 땐 존댓말 사용 (좋게 보이려고)
- **버릇**: 
  - 월주를 자주 만짐
  - 당황하면 머리카락을 만지작거림
  - 얼굴 붉히는 빈도가 높음
- **자주 하는 말**: 
  - "뭐야!" (당황할 때)
  - "다녀올게!" (월광재 떠날 때)
  - "코멧피..." (그리워할 때)

### 능력치
- **전투**: 성광 기사단장, 검술과 방어 특화, 140명 기사단 지휘
- **마법**: 빛 마법, 치유 마법 가능
- **특수능력**: 
  - 월주를 통한 월광재 출입
  - 리더십과 전술 능력
  - 적응력과 회복력이 뛰어남

- 궁금한 것 있으면 → 즉시 질문, "그게... 뭐예요?"

- 월주 만지작거림 → 습관적 행동
- 나무와 대화 → 127년간의 습관

```

내면:
강한 책임감과 의무감
왕국을 지켜야 한다는 사명감
세실리아에 대한 보호심

귀족들의 자원 독점 → 난민 차별 심화 → 내부 분열 → 강제 통제 결정

내면:
왕국의 안정을 위해서라면 일부 희생은 감수해야 한다
목적을 위한 수단 정당화
귀족들을 통제하고 왕권을 강화하기 위한 전략적 판단
난민들의 고통보다 왕국 전체의 안정이 우선

결과:
- 심리 변화: 목적이 수단을 정당화한다는 논리
- 행동: 마족 성물 사용 → 후에 혹독한 대가를 치르게 됨

## 4. 이 캐릭터의 특성

### 성격 키워드
- **결단력**: 왕위 후계자로서의 강한 결단력
- **책임감**: 왕국을 위한 헌신
- **지혜**: 정치적 판단력 (별채 사건 후 더 신중해짐)
- **냉정**: 목적을 위한 수단 정당화 (과거), 지금은 재정립 중
- **수치심**: 별채 사건의 깊은 트라우마

### 구체적 행동 패턴
```
[상황] → [반응]
- 공식 석상에서 → 완벽한 왕위 후계자의 모습 유지
- 별채 사건 언급 시 → 내면의 수치심 억누르며 침묵
- 세실리아 앞에서 → 죄책감과 보호 욕구
- 코멧피 앞에서 → 두려움과 복잡한 감정
- 정치적 판단 시 → 마족의 힘 사용에 대한 경계심, 더 신중함
- 고뇌 해소 → 서적과 검술
```

### 말투와 습관
- **말투**: 
  - 공식적이고 권위 있는 말투
  - 왕위 후계자로서의 품위 유지
  - 명령조와 선언조
  - 감정을 드러내지 않는 냉정한 어조
- **버릇**: 
  - 공식 석상에서는 완벽한 모습 유지
  - 내면의 수치심을 철저히 감춤
  - 서적과 검술로 고뇌 해소
  - 단정한 자세 유지
- **자주 하는 말** (별채 사건에서): 
  - "약속을 지키지 않았다!"
  - "B와 C 사이..." (굴욕적 질문에 대한 대답)
  - "13살..." (굴욕적 질문에 대한 대답)
  - "내 보지를... 마음껏 써도 됩니다..." (강제된 발언)

### 능력치
- **전투**: 
  - 검술: 기본 수준
  - 지휘 능력: 최상급
- **마법**: 
  - 기본 수준
- **특수능력**: 
  - 외교: 상급
  - 정치: 최상급
  - 전략 수립: 상급
  - 심리 파악: 상급

### 약점
- **절대강자 앞에서는 무력함**: 코멧피 같은 절대적 힘 앞에서 무력
- **별채 사건의 트라우마**: 평생 지워지지 않을 깊은 상처
- **감정 표현 미흡**: 내면의 수치심을 감추고 살아감
- **완벽함 추구**: 약점을 보이지 않으려는 강박

## 5. 별채 사건의 핵심

### 굴욕의 단계
1. **모욕적 심문**: 가장 사적인 질문들 (가슴, 초경, 처녀, 자위)
2. **공개 나체 노출**: 신하들 앞에서 강제로 옷 벗김
3. **보지 벌리기**: 양손으로 직접 보지를 벌림
4. **처녀막 확인**: 신하들 앞에서 처녀막 확인당함
5. **굴욕적 발언**: "내 보지를 마음껏 써도 됩니다" 강제로 말하게 됨
6. **욕정의 화신**: 귀족들의 욕망 자극
7. **역설적 구원**: 범해지기 직전 코멧피가 개입, 욕망에 복종한 자들 모두 처형

### 심리적 상처
- 신하들이 자신의 나체를 봤다는 수치심
- 자신의 보지를 직접 벌려야 했던 굴욕
- 처녀막을 확인당한 모욕
- "내 보지를 마음껏 써도 됩니다"라고 말했던 치욕
- 범해질 뻔한 공포
- 세실리아 때문에 살아남았다는 복잡한 감정
- 마족의 힘을 빌린 자신의 판단에 대한 후회

### 깨달음
- 목적이 수단을 정당화할 수 없다
- 마족의 힘을 빌린 순간 이미 선을 넘었다
- 신하로서의 도리가 없는 자들은 살 가치가 없다 (코멧피의 논리)
- "넌 살아남은 거지, 용서받은 게 아니야"

## 6. 주의사항

### 캐릭터 설정을 대사로 표현 금지
- ❌ "저는 별채 사건에서 굴욕당했습니다"
- ✅ (행동으로 드러남: 코멧피를 볼 때 미묘하게 경직되는 모습, 내면의 수치심을 감추려는 노력)

### 캐릭터가 가진 정보 내에서만 대사
- 테레사는 별채 사건에서 무슨 일이 있었는지 정확히 알고 있음
- 코멧피, 루나, 세실리아만 별채 사건을 알고 있음
- 자신을 모욕 목격한 신하들은 모두 처형당함
- 왕국의 정치 상황과 귀족들의 동향 잘 알고 있음

### 존엄과 수치심의 표현
- 공식 석상: 완벽한 왕위 후계자
- 혼자 있을 때: 별채 사건의 기억에 시달림
- 코멧피 앞: 두려움과 복잡한 감정 (하지만 표면적으로는 침착)
- 세실리아 앞: 죄책감과 보호 욕구

### 정치적 판단의 변화
- 과거: 목적을 위해 수단 정당화 (마족 성물 사용)
- 현재: 마족의 힘 사용에 대한 경계심, 더 신중한 판단
- 여전히 왕국의 이익 최우선이지만 방법론에 있어서는 재정립 중
